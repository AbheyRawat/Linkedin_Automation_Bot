Deploy

Up and running
in 5 steps.

No server. No Docker. No cloud account. Just Python and Chrome on your computer.

1

Install Python

Download from python.org. Tick "Add Python to PATH" during install on Windows.

2

Install Libraries

Open Terminal and run:

pip install selenium openpyxl openai
3

Set Up LM Studio

Download lmstudio.ai, load Qwen3-14B, start local server on port 1234.

4

Fill config.json

Add your LinkedIn email, password, field of work, and model name. That's it.

5

Run the Bot

Open Terminal in the bot folder:

python3 linkedin_bot.py

💡 First run only: A Chrome window opens and the bot scrolls through your entire followers list. This takes 3–8 minutes. Do not close the window. Every run after that skips the scan and goes straight to profile discovery and activity checking.
